Memory Management Framework

A C++ memory management framework inspired by PostgreSQL's memory context system — hierarchical contexts let you allocate from active contexts, switch between them, reuse freed chunks, and recursively tear down entire context trees. Each allocation stores metadata for ownership tracking with 8-byte aligned payloads, retaining released chunks for reuse until the owning context is deleted. C++ templates add typed object construction/destruction for RAII-aware lifecycle management on top of raw allocation.
